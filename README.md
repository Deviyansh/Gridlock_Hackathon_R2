# Gridlock: AI-Powered Traffic Incident Management System

## Overview

Gridlock is an AI-powered traffic incident prioritization and response management platform designed for smart cities.

The system predicts traffic incident priority levels, assesses severity, estimates traffic impact, recommends response actions, and assists traffic authorities in allocating resources effectively.

The solution is demonstrated using Bengaluru traffic operations scenarios.

---

## Problem Statement

Traffic management authorities face significant challenges in:

- Identifying critical incidents quickly
- Prioritizing response efforts
- Allocating resources efficiently
- Reducing congestion caused by accidents and disruptions
- Coordinating traffic operations in real time

Gridlock addresses these challenges using machine learning and decision-support analytics.

---

## Features

### AI Incident Priority Prediction

Predicts whether an incident requires:

- Priority 1 (Critical)
- Priority 2 (Medium)
- Priority 3 (Low)

### Severity Assessment

Generates severity scores based on:

- Event cause
- Event type
- Location
- Time of occurrence
- Road closure requirements

### Traffic Impact Estimation

Estimates:

- Local Impact
- Zone-Level Impact
- City-Level Impact

### Resource Allocation

Recommends:

- Traffic Officers
- Barricades
- Emergency Resources

### Response Recommendation Engine

Suggests:

- Traffic Police deployment
- Emergency Services
- Ambulance Support
- Traffic Diversions
- Public Advisory Actions

### Live Operations Center

Provides:

- Active incidents overview
- Traffic status monitoring
- Resource deployment visibility

### Interactive Bengaluru Traffic Map

Visualizes traffic operations across:

- Central Bengaluru
- North Bengaluru
- South Bengaluru
- East Bengaluru
- West Bengaluru

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/gridlock.git
```

Move into the project directory:

```bash
cd gridlock
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Running the Application

```bash
streamlit run app/dashboard.py
```

Open the URL generated by Streamlit in your browser.
